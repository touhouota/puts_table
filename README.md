# putsで表のように表示させたい
同じ要素を持つArrayの中身をSQLのテーブルのような見た目で出力させるやつ。

例：
```
require "./lib/puts_table"

array = [
  {
    a: 1,
    b: 12,
    c: 123
  },
  {
    a: 11111,
    b: 23234243,
    c: 22423423424342
  }
]

puts_table array
# +-------+----------+----------------+
# | a     | b        | c              |
# +-------+----------+----------------+
# | 1     | 12       | 123            |
# | 11111 | 23234243 | 22423423424342 |
# +-------+----------+----------------+
```
